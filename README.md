# apartment-ai-price-app

아파트·근린상가 분양 및 거래가액 분석 앱

## 설치 및 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## Vercel 설정값

- Framework Preset: Vite
- Root Directory: ./
- Build Command: npm run build
- Output Directory: dist
