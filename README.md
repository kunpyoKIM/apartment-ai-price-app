# apartment-ai-price-app-final-green-select-style

아파트·근린상가 분양 및 거래가액 분석 앱

## 변경사항

- 선택 버튼을 큰 둥근 버튼, 아이콘 강조, 그림자 스타일로 변경함.
- 선택됨: 초록색 바탕 + 진한 글씨
- 미선택: 흰색 바탕 + 진한 글씨, 다크모드에서는 진회색 바탕 + 흰색 글씨
- 전체 초록색 테마 유지함.
- Vercel 배포 오류 방지를 위해 `tailwindcss 3.4.17`로 고정함.

## 설치 및 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## Vercel 설정값

- Framework Preset: Vite
- Root Directory: ./
- Build Command: npm run build
- Output Directory: dist
